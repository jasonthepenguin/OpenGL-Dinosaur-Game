# OpenGL-Dinosaur-Game

My first project exploring the idea of creating a basic game engine with C++, Modern OpenGL, Lua

- Basic Lua scripting
- Basic AI NPCs
- Basic game loop
- Testing ReactPhysics3D for projectiles
- Loading MD2 models and their animations
